**Lab 07 : Responsabilisez votre personnel – Copilot – RH**

**Objectif :**

Avec Copilot pour Microsoft 365, les professionnels des RH peuvent
rationaliser leur flux de travail et améliorer leur productivité dans
leurs processus métier de base tels que le recrutement, l'intégration,
la gestion des performances, l'engagement des employés et la gestion de
la conformité.

Dans cet atelier, vous allez utiliser :

- Copilot dans Word pour créer une description de poste pour un nouveau
  rôle.

- Copilot dans Word pour analyser plusieurs CV et fournir un rapport
  comparant les forces et les faiblesses de chaque candidat, classer les
  candidats du plus qualifié au moins qualifié, et faire une
  recommandation.

- Copilot in Loop pour créer un ensemble de questions d'entretien pour
  interviewer les candidats à ce poste.

Word**

concepteur d'animation senior. Votre personnel a créé un document qui

vous allez utiliser Copilot dans Word pour créer une description de
poste basée sur les responsabilités du rôle dans ce document.

**Exercice \#2 : Analyser des CV et faire une recommandation à l'aide de
Copilot dans Word**

Dans l'exercice précédent, vous avez appris comment Copilot dans Word
peut aider les professionnels des RH à rédiger une description de poste.
Dans cet exercice, vous allez apprendre comment automatiser les
premières étapes du processus de sélection des CV afin d'identifier
rapidement les candidats les plus appropriés parmi de grands bassins de
candidats.

**Remarque** : Lorsque vous demandez à Copilot de générer un document ou
d'appliquer un certain type de modification, il peut parfois commencer à
afficher le brouillon puis s'arrêter. Si cette situation vous arrive,
sélectionnez le bouton **Regenerate** pour qu'il génère un nouveau
brouillon, ou reformulez votre invite et réessayez.

Institute, vous avez commencé à recevoir des CV de candidats potentiels

![](./media/image16.png)

**Exercice \#3 : Créer des questions d'entretien à l'aide de Copilot in
Loop**

Avec Copilot in Loop, vous pouvez créer des espaces de travail et des
pages, utiliser une recherche intelligente et des modèles pour ajouter
du contenu pertinent et partager votre travail avec d'autres. Copilot in
Loop peut suggérer des idées et vous aider à démarrer des projets, ce
qui vous permet de démarrer plus facilement lorsque vous vous retrouvez
bloqué. Il peut rédiger des textes, créer des tableaux et même répondre
rapidement à des questions.

Copilot in Loop peut aider les professionnels des RH à travers de
nombreux aspects de l'expérience RH - du processus d'embauche à la
gestion des employés, en passant par la gestion des documents importants
et la gestion des communications internes.

Dans cet exercice, vous allez utiliser Copilot in Loop pour faciliter le
processus d'embauche pour un nouveau rôle. Alors que vous avez utilisé
Copilot dans Word dans l'exercice précédent pour filtrer les CV, vous
allez utiliser Copilot in Loop pour créer une liste de questions
d'entretien pour les meilleurs candidats. Au cours de cet exercice, vous
pouvez vérifier si Copilot in Loop peut modifier la liste de questions
en fonction de votre direction.

1.  Si **Microsoft 365** est ouvert dans un onglet, passez à l'étape
    suivante. Sinon, ouvrez un nouvel onglet dans votre navigateur
    **Microsoft Edge**, puis entrez l'URL suivante :
    +++[https://www.office.com+++](https://www.office.com+++/)

2.  Dans **Microsoft 365**, sélectionnez **Loop** s'il apparaît dans le
    volet de navigation de gauche. S'il n'apparaît pas dans le volet de
    navigation, sélectionnez **App Launcher**, puis sur la page
    **Apps**, faites défiler vers le bas pour localiser **Loop,** puis
    sélectionnez-le.

![](./media/image18.png)

3.  Si vous voyez le bouton **Sign-in,** connectez-vous à l'aide des
    informations d'identification de l'utilisateur.

![](./media/image19.png)

**Remarque** : Une fois connecté, fermez la fenêtre du navigateur Loop
et ouvrez à nouveau Loop à partir de la page Applications.

4.  Dans **Microsoft Loop**, l'onglet **Workspaces** s'affiche par
    défaut. Vous souhaitez créer un nouvel espace de travail pour ce
    projet, sélectionnez **+** en regard de **Getting started**, puis
    sélectionnez le bouton **+New workspace**.

![](./media/image20.png)

5.  Dans la fenêtre **Create a new workspace**, saisissez **Interview
    questions** pour le nom de votre espace de travail, puis
    sélectionnez le bouton **Continue** ou **Create** qui s'affiche
    (selon votre version de Loop).

6.  Dans la fenêtre **Add files to your workspace** (selon votre version
    de Loop, cette fenêtre peut indiquer **Workspace Switcher**),
    sélectionnez **Create a new workspace**.

7.  Vous êtes maintenant sur la première page de votre nouvel espace de
    travail. Le nom de la page est actuellement **Untitled**. Notez que
    la page (Sans titre) apparaît également dans le volet de navigation
    de gauche. Dans le corps principal de la page, sélectionnez dans le
    champ **Untitled** et remplacez le nom de la page par : **15
    interview questions for the Senior Animation Designer role**. Notez
    que le nom de la page est automatiquement mis à jour dans le volet
    de navigation.

![](./media/image21.png)

8.  Dans la section **Just start typing...** , entrez une barre oblique
    **(/).**

![](./media/image22.png)

9.  Dans le menu déroulant qui s'affiche, sous la section **Copilot** en
    haut du menu, sélectionnez **Brouillon de contenu de la page**.

10. Dans la fenêtre **Copilot** qui s'affiche, entrez l'invite suivante
    et sélectionnez l' icône **Submit** :

++**Create a list of the 15 best interview questions that should be
asked to candidates applying for a new Senior Animation Designer role at
the Graphic Design Institute**.++

![](./media/image23.png)

**Remarque** : Il se peut que vous ne voyiez pas la fenêtre **Copilot**
avec les options Créer, Brainstormer, Blueprint et Décrire. Si vous
rencontrez de telles exceptions, fermez l'espace de travail et
réessayez.

11. Passez en revue la liste de questions.

![](./media/image24.png)

12. Vous pensez que cette liste initiale est un bon début, mais vous
    remarquez que quelques types de questions manquent. Dans le volet
    Copilot qui s'affiche, entrez l'invite suivante :

learned from it**.++

![](./media/image25.png)

![](./media/image26.png)

    leadership. Pour corriger cette situation, entrez l'invite suivante

their design team on projects. Ask them to talk about a couple of their

![](./media/image27.png)

15. Notez ce qui s'est passé dans Loop. Lorsque vous demandez une
    modification sans lui demander spécifiquement de mettre à jour la
    page actuelle, Copilot peut ouvrir une nouvelle page et y apporter
    la modification plutôt que dans la page actuelle, comme il l'a fait
    dans l'invite précédente. Lorsqu'il le fait, notez que dans le volet
    Copilot, il suit les invites, l'invite la plus récente apparaissant
    en haut du volet.

Sélectionnez une invite précédente. Notez que le contenu de la page
s'affiche dans une police plus claire. Essayez également de sélectionner
le champ d’invite **Rewrite with Copilot**. Copilot désactive ce champ
car il ne s'agit pas de la page actuelle. La page active est également
appelée page active. Sélectionnez maintenant la dernière invite et notez
que le contenu est clair, indiquant que cette page est la page actuelle

sur cette page, vous pouvez sélectionner le champ d'invite **Rewrite

![](./media/image28.png)

16. Faisons un changement de plus. Vous n'êtes pas sûr que la liste
    actuelle de questions soit suffisante. Pour jouer la sécurité, vous
    décidez de demander à Copilot in Loop de vous recommander quelques
    questions supplémentaires. Entrez l'invite suivante :

++**Are there any other questions that you think should be added to the
list**?++

17. Notez la liste finale des questions générées par Copilot in Loop.

![](./media/image29.png)

18. Vous disposez désormais d'un vaste assortiment de questions parmi
    lesquelles choisir lorsque vous interviewez vos candidats. Et avec
    Loop, vous savez que d'autres membres de votre équipe RH peuvent
    voir ces questions en temps réel sur le même composant Loop, même
